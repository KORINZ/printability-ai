# AI-Powered Printability Evaluation Framework for 3D Bioprinting Using Hausdorff Distance Metrics

This repository contains scripts used for the paper "AI-Powered Printability Evaluation Framework for 3D Bioprinting Using Hausdorff Distance Metrics." The paper is currently under review.

**Keywords:** 3D bioprinting; Hausdorff distance; printability; cell viability; machine learning; convolutional neural network

## Disclaimer

Please note that some scripts in this repository might require the corresponding data files to run successfully. The data files are available upon reasonable request from the corresponding author.
